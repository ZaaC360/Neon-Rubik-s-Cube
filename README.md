##  How to Break It

**Keyboard Magic:**
- `W` / `S` → Spin up/down  
- `A` / `D` → Spin left/right  
- `F` / `B` → Spin front/back  
- Hold `Shift` for reverse spins

**Mouse:** Just drag it around like the cube owes you money
